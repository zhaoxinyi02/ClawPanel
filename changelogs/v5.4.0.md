# ClawPanel Pro v5.4.0

发布时间：2026-04-08

## OpenClaw 2026.4.8 对齐

- 面板整体同步到 OpenClaw `2026.4.8` 语义，补齐了原生任务板与 Dashboard `task-pressure` 摘要
- 通道管理补齐更多 OpenClaw 内置通道，并支持按上游 `configSchema / uiHints` 动态渲染未知通道的基础表单
- 统一 pairing 审批入口，减少用户在 CLI 和面板之间来回切换

## 系统配置体验整理

- 身份页重新整理布局：登录与修改密码入口合并，Markdown 身份文档编辑区层次更清晰
- Agent 默认设置改为弹窗编辑，避免高密度参数长期占用主页面
- 系统配置页各卡片展开状态支持本地持久化，下次打开仍保留上次视图

## 通道与工作流补强

- 微信 `openclaw-weixin` 通道继续完善，配套状态与登录能力已整合进面板
- 工作流、通道状态、活动日志、搜索与 Cron 配置继续向 OpenClaw 当前能力对齐
