# ClawPanel Lite v0.1.13

发布时间：2026-03-18

## 自定义技能市场与配置体验增强

- 自定义技能仓库接入后的 `skill.json` 结构化配置现在可以直接在面板中编辑与保存
- 已安装技能支持复制到其他智能体或全局目录，便于多智能体复用技能
- 技能市场安装 / 卸载后的刷新与状态识别继续补强，减少已安装状态不同步

## 通道与前端体验优化

- 合并飞书多 ID 兼容增强，统一兼容 `openclaw-lark` 与旧官方 ID
- 前端性能优化继续下放到 Lite：首屏拆包、隐藏标签页暂停轮询、Channels 初始化并行加载
- 模型配置页同步获得更完整的高级配置 UI

## 说明

- Lite 当前版本号更新为 `0.1.13`
- 本次发布主要聚焦技能市场、配置体验与合并后的稳定性收口
