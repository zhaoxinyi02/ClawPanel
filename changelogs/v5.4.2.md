# ClawPanel Pro v5.4.2

发布时间：2026-04-11

## 通道管理体验优化

- QQ 官方机器人配置页新增「内置版 / 社区版」切换，切换后自动启用当前版本并关闭另一个版本
- QQ 官方机器人配置收敛为主配置优先（App ID / App Secret），其余字段保留在高级配置中按需展开
- 通道管理默认不再强制选中某个通道，进入页面先展示引导占位视图，减少误操作

## 活动与会话可见性修复

- 修复最近活动吞消息问题，补齐会话回放中 assistant 回复的解析与展示
- QQ / 微信通道活动统一做来源归一化，减少同一通道在不同页面被拆分显示
- 活动日志增加低价值分类折叠（heartbeat / agent），默认突出主要会话流

## 稳定性与部署改进

- 修复活动日志按类别筛选后其它类别消失的问题，筛选链路改为先检索后分类
- 本地构建产物统一迁移到 `local-binaries/`，并加入 Git 忽略，避免误提交二进制文件
