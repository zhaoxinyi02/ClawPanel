# ClawPanel Pro v5.3.4

发布时间：2026-04-06

## 本机更新镜像

- 面板自更新与独立更新页改为先同步 GitHub Release 元数据和二进制到本机缓存，再从本机缓存完成替换
- 新增本机更新镜像接口，统一暴露 `update-mirror` 元数据与平台构建包下载入口
- 更新流程现在强制要求存在 `checksums.txt` 并完成 SHA256 校验，不再接受“无校验继续更新”

## 公开脚本入口统一

- 新增公开脚本/插件/二进制资源路由，安装、更新、修复、卸载脚本都可以直接从面板本机入口下载
- 安装与更新脚本统一收口到 `CLAWPANEL_PUBLIC_BASE`，后续迁移服务器或切换域名时无需再批量替换脚本内地址
- 文档中的安装、更新与修复示例已同步改为 `CLAWPANEL_PUBLIC_BASE` 风格

## 稳定性修复

- 修复独立更新服务在停服后仍依赖外部下载链路，导致更新过程中因网络抖动失败的问题
- 将插件注册表镜像、QQ 插件辅助下载与 NapCat 修复链路一并切到本机优先，保留 GitHub 兜底
