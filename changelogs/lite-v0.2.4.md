# ClawPanel Lite v0.2.4

发布时间：2026-04-06

## 本机更新镜像

- Lite 更新链路改为先同步 GitHub Release 元数据和整包到本机缓存，再从本机缓存执行替换
- 新增本机更新镜像接口，统一暴露 Lite 构建包下载入口
- 更新流程现在强制要求存在 `checksums.txt` 并完成 SHA256 校验，不再接受“无校验继续更新”

## 公开脚本入口统一

- Lite 安装、更新、修复、卸载脚本已经统一挂到面板本机公开入口
- 安装与更新脚本统一收口到 `CLAWPANEL_PUBLIC_BASE`，后续迁移服务器或切换域名时无需再修改多处地址
- 文档中的 Lite 安装、更新与卸载示例已同步改为 `CLAWPANEL_PUBLIC_BASE` 风格

## 稳定性修复

- 修复更新过程中停服后再访问外部下载源导致链路中断的问题
- NapCat / QQ 插件修复脚本改为本机优先，保留 GitHub 归档兜底
