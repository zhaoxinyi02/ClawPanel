# ClawPanel Pro v5.3.0

发布时间：2026-03-29

## 登录与界面体验

- 统一登录页与主面板的视觉语言，重做透明 Logo、玻璃质感卡片与整体比例
- 登录页支持密码框回车直接登录，并调整底部品牌文案
- 仪表盘在线状态、最近活动、离线提示等页面体验继续收口，减少重复告警与误导信息

## OpenClaw / 通道稳定性修复

- 修复插件安装后 OpenClaw 未自动重启、插件变更后运行态恢复不及时的问题
- 修复 QQ 官方机器人等插件通道的安装链路，优先走官方包安装并补强网络抖动场景
- 修复面板聊天错误继承 `channels/plugins` 运行时配置，避免出现 `unknown channel id: qqbot`
- 修复频道页、首页与最近活动对插件通道状态的误判与脏状态残留

## 运行与可观测性

- 最近活动扩大单会话预览窗口，并过滤审批回调类内部消息，减少“吞消息”和幽灵日志
- 去掉页面内重复渲染的 OpenClaw 离线提示，只保留全局顶部告警
- 继续补强插件安装测试、面板聊天运行时测试与相关回归覆盖
