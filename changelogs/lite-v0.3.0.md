# ClawPanel Lite v0.3.0

发布时间：2026-04-08

## Lite 运行时升级

- Lite 内置 OpenClaw runtime 默认版本提升到 `2026.4.8`
- 新装后即可直接使用更新后的 OpenClaw 能力，减少首次部署后继续手动升级运行时的成本

## OpenClaw 2026.4.8 对齐

- 面板同步补齐原生任务板、Dashboard `task-pressure` 摘要、统一 pairing 审批入口，以及扩展后的内置通道目录
- 未手写适配的新通道也能先通过 schema 动态表单在面板中显示、保存与启停

## 配置中心整理

- 身份页和通用配置页重新整理布局，降低卡片拼贴感
- Agent 默认设置改为弹窗编辑，配置块展开状态支持本地持久化
