# ClawPanel Pro v5.2.13

发布时间：2026-03-18

## 自定义技能市场继续完善

- 新增 `ClawPanel-Plugins` 自定义技能仓库接入后的结构化配置能力，`skill.json` 里的配置字段现在可以在面板里直接渲染成表单
- 技能页支持读取并保存自定义技能的 `configSchema`，覆盖文本、密码、多行文本、下拉、开关与数字等常见字段类型
- 自定义技能复制能力补齐，已安装技能现在可以复制到其他智能体或全局目录，不再误走商店安装路径

## 技能与通道体验增强

- 合并飞书多 ID 兼容增强，统一兼容 `openclaw-lark` 与 `feishu-openclaw-plugin`
- Skills 页补齐插件批量启用 / 禁用 / 卸载、商店卸载刷新与安装目标修正
- 自定义技能市场继续收口，模板目录与超大文件技能不再错误暴露为可安装项

## 前端与系统配置优化

- 合并前端性能优化：首屏 vendor 拆包、隐藏标签页轮询暂停、Channels 页面并行加载
- 模型配置页升级为更紧凑的高级模型配置 UI，支持 provider / model 自定义 headers、compat 标记、cost 字段等配置
- 侧边栏运行中通道排序与 `+N` 显示逻辑修正
