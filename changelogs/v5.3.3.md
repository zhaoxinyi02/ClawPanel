# ClawPanel Pro v5.3.3

发布时间：2026-03-31

## 企业微信（智能机器人）双模式配置

- 同一页面支持 `URL 回调` 与 `长连接` 两种连接方式切换，不再拆成多个页面
- URL 回调模式填写 `Token / EncodingAESKey / Webhook Path`
- 长连接模式填写 `Bot ID / Secret`，保存时自动规范化到 `channels.wecom`

## 官方插件通道安装热修

- `openclaw plugins install` 遇到 `ClawHub 429` 限流时，自动回退到 `npm pack` 本地 tarball，再交给 OpenClaw 安装
- 继续保留官方命令优先、npm 兜底的安装顺序，减少国内网络抖动与限流造成的安装失败

## 企业微信插件 ID 兼容修复

- 兼容 `wecom-openclaw-plugin` 与历史 `wecom` 插件 ID
- 修复企业微信插件安装后面板识别、启用/禁用和保存配置仍写旧 entry 的问题
