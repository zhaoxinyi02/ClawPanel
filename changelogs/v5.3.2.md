# ClawPanel Pro v5.3.2

发布时间：2026-03-31

## 面板聊天

- 新增面板聊天能力，支持直接在面板内与本地 OpenClaw 对话，继续使用工作区、技能与本地文件上下文
- 支持单 AI 对话与多 AI 顺序群聊，可为新会话选择参与 Agent 与总结 Agent，适合日常调试、代码协作与任务拆解
- 支持会话重命名、删除、代码块复制、中途停止与失败后恢复输入，减少来回切换原生 CLI 的成本

## 官方插件通道安装链路修复

- QQ 官方机器人、飞书、钉钉、企业微信等官方插件通道不再强依赖远端仓库命中
- 安装时优先执行 `openclaw plugins install`，失败后再回退 `npm install -g`，避免因 registry 刷新失败而直接报“插件不在仓库中”

- 更新 Lite / Pro 最近版本展示、更新元数据与 README 社区入口说明
- 精简 README 中的 GitHub Actions 自动化说明块，并补充面板聊天能力介绍
